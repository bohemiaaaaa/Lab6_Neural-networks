# Лабораторная работа №6: Сравнительный анализ архитектур нейронных сетей для задач регрессии в PyTorch и TensorFlow

## Цель работы

Изучить и применить методы построения, обучения и оценки нейронных сетей для решения задач регрессии с использованием фреймворков PyTorch и TensorFlow; провести сравнительный анализ различных архитектур нейронных сетей и их влияния на качество предсказаний на примерах прогнозирования заработной платы, количества фолов в баскетбольных матчах и стоимости автомобилей.

## Задачи работы

### Задание 1 (PyTorch + TensorFlow)
На основе подготовленной таблицы данных с HeadHunter создать 6 архитектур нейросетей, используя только числовые данные (без текстовых).

| № | Архитектура | Описание |
|---|-------------|----------|
| 1 | Базовая | 1 скрытый слой (64 нейрона) |
| 2 | Глубокая | 4 скрытых слоя (128→256→128→64) |
| 3 | Dropout | Добавлены Dropout слои для регуляризации |
| 4 | BatchNormalization | Добавлены слои нормализации |
| 5 | Узкая | 3 скрытых слоя (32→16→8) |
| 6 | LR Annealing | Адаптивное уменьшение learning rate |

### Задание 2 (PyTorch + TensorFlow)
На базе баскетбольных матчей разработать нейросетевую модель для предсказания количества фолов.
- **Целевой результат:** средняя абсолютная ошибка (MAE) ≤ 17 очков

### Задание 3 (PyTorch + TensorFlow)
По базе данных машин с Юлы обучить модель для предсказания цен на автомобили.

**Требования:**
1. Создать обучающую, тестовую и проверочную выборки
2. Оценить качество модели, определив средний процент ошибки на проверочной выборке
3. Подсчитать ошибку на каждом примере тестовой выборки и суммарный процент ошибки

**Рекомендации:**
- Функция потерь: MSE (среднеквадратическая ошибка)
- Последний слой модели — 1 нейрон
- Суммарный процент ошибки = MAE / средняя цена машины × 100%

## Технологии

- **Фреймворки:** PyTorch, TensorFlow / Keras
- **Библиотеки:** NumPy, pandas, scikit-learn, Matplotlib, gdown, re
- **Типы слоёв:** Linear, BatchNorm1d, Dropout
- **Оптимизаторы:** Adam, ReduceLROnPlateau
- **Функции потерь:** MSELoss (PyTorch), MSE (TensorFlow)
- **Метрики:** MAE, MSE
